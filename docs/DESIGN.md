# Design System Document: The Forensic Architect

## 1. Overview & Creative North Star
The 'Forensic Architect' is a design system engineered for the deep-tissue analysis of legacy COBOL systems. It rejects the 'general-purpose dashboard' aesthetic in favor of a **High-Density Technical Environment**. 

**The Creative North Star: 'Precision Noir'**
This system treats code as an archaeological site. It moves beyond standard UI by utilizing **Tonal Layering** and **Intentional Asymmetry**. Instead of a rigid, centered grid, we use a 'Weighted Workbench' layout where information density is balanced by deep, atmospheric voids.
